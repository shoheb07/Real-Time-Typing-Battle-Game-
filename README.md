# Real-Time-Typing-Battle-Game-
Real-Time Typing Battle Game is a multiplayer typing competition where players join a room and compete in real time. The player with the highest WPM (Words Per Minute) and accuracy wins.


A multiplayer typing speed competition game built using Node.js, Express, and Socket.io.
